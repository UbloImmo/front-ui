## Usage

VirtualTable is a layout component that renders a virtualized table list.
It relies on the [react-virtuoso](https://virtuoso.dev/) package to achieve this.

> Virtualized rendering is a technique that renders only the visible items in a list,
> and only pre-renders the items that are close to the currently visible items.
> This is particularly useful for rendering large (10K+ items) lists, as it can significantly reduce the amount of memory used.

### Rendering data

VirtualTable relies on two properties to render

#### Data

The `data` property expects an list of objects items to render. Each item in the list will be rendered as a row in the table.

> An empty state may be shown if the data array is empty by providing the `EmptyState` property.

#### Columns

The `columns` property is used to define what to render for each item in the array.

Each column takes at least a `HeaderContent` label, object or function component (used to render the column's header), and a `CellContent` function component - used to render its cell for each row.

> Take a look at each example's source code for code examples.

##### HeaderContent

Defines the content to be rendered in a column's header.
