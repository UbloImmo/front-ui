## Usage

`<FlexLayout>` renders by default with direction property as `row`, justify and align properties as `start`.
Its properties are based on the CSS rules for flexboxes (`wrap`, `align`, `reverse`, etc.).

### Row

`<FlexRowLayout>` can also be used to render a flex layout with a `row` direction. In the example below, `justify` is set to **space-between**
