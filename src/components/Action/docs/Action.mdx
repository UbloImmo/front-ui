## Usage

### Sizing

Action provides 2 sizes (`default` & `large`) to help users differenciate between multiple actions.

<br />
*As expected, the default size is `default`.*

> Actions are set to fill their container width-wise (`width: 100%`).

<Canvas of={ActionStories.Sizes} />

### Basic customization

Action buttons can have their `label`, `icon` and `badgeLabel` properties customized to fit most needs.

#### Label

There are no hard limits to the Action's label. Be wary of very long labels as they could be clipped on smaller window sizes.

> Action labels are allowed to wrap up to 2 lines before getting clipped.

<Canvas of={ActionStories.Labels} />

#### Icon

Actions internally use a [StaticIcon](?path=/docs/components-staticicon-usage--docs) to render their Icon.
As such, they can take any valid `IconName`.

> Please refer to the [Icon Repository](?path=/docs/components-icon-repository--docs) for a complete list.

<Canvas of={ActionStories.Icons} />

### Further customization

#### Badge

Provide a `badgeLabel` to display a [Badge](?path=/docs/components-badge-usage--docs) alongside the Action's label.
Its color will change between `primary` and `gray` depending on the Actions's `disabled` property.

> No `Badge` will be rendered if `badgeLabel` is an empty string or missing.

<Canvas of={ActionStories.BadgeLabels} />

#### Tooltips

Add a [Tooltips's properties object](?path=/docs/components-tooltip-properties--docs) as its `iconTooltip` property to display a tooltip inside the action.

> Any children passed to the `iconTooltip` property will be ignored so as to always render an icon

<Canvas of={ActionStories.Tooltip} />

#### Description

Use the `description` property to display additional text below the action's label, providing users with more context or details about the action's purpose.

<Canvas of={ActionStories.Description} />

#### Indicator

Use the `indicator` property to display an additional element on the top right corner of the static icon.

<Canvas of={ActionStories.Indicator} />

### Disabled

Actions are inherently buttons with a bit of _flair_. Therefore, they can be disabled.

<Canvas of={ActionStories.Disabled} />
