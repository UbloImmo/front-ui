## Usage

`Collapsible` is a component that functions similarly to an accordion, allowing users to expand and reveal its nested content by clicking on the caret icon (<Icon name="CaretRightFill" color="gray-900" size="s-2" />).

`Collapsible` components are nested to create a tree-like structure of expandable content.

> Which means that you can nest `Collapsible` components as many levels deep as you need 🤯.

<Canvas of={CollapsibleStories.Default} />

## Custom content

`Collapsible` is a flexible component that accepts any item or component as its `children`, allowing you to fully customize both the main content and any sub-elements.

<Canvas of={CollapsibleStories.CustomChildren} />

## Compact

You can change the display of `Collapsible` to a compact layout by setting the optional `compact` property.
