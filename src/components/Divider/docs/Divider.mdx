## Usage

A divider is a single, non-interactive horizontal line spanning the whole width of its container

### Label

By providing a `label` property, a divider can display it to the left of its line.

> Divider labels get rendered as bold `h4` [Headings](?path=/docs/components-typography-heading-usage--docs).

<Canvas of={DividerStories.Labels} />

### Label alignment

Label's default alignment in the Divider is on the left of its line with the property `labelAlign` set to `start`.

A variant is available to center the label in the Divider between two lines.
To use it, set `labelAlign` to `center`.

> In this variant, the label is rendered in a smaller size, as a medium `s` [Text](?path=/docs/components-typography-text-usage--docs)
