You can use one of the 3 preset sizes according to your needs.

If no size is provided, defaults to `m`

### Font weight

Control your text's weight with the `weight` property. Defaults to `regular`.

<Canvas of={TextStories.Weights} />

### Italics

If you need to render italic text (for some obscure reason), set this component's `italic` property to `true`.

It defaults to `false`, obviously.

<Canvas of={TextStories.Italic} />

### Decorations

Need to spice up your text ?
Slap an <Text underline>underline</Text>, <Text overline>overline</Text> or a <Text lineThrough>line through</Text> it.

Each decoration maps to its own property (`underline`, `overline`, `lineThrough` respectively).

Set some - or all - of them to `true` to see the result.
Defaults to `false`.

<Canvas of={TextStories.Decorations} />

### Colors

Colorize a text block's color by passing it any valid color variable.

See our [Color Palette](?path=/docs/color-palette--docs)'s documentation for an exhaustive list of usable values.
