## Usage

### Example

When using multiple `ContextLine` components, their padding and box-shadow will adapt according to their position in the parent component (first-child, default and last-child).
You can inspect it to see the effect.

<Canvas of={ContextLineStories.Example} />

### Label

A label must be provided in the `label` property. It's a required property.

<Canvas of={ContextLineStories.Label} />

#### Description

A optional description may be provided through the `description` property.

<Canvas of={ContextLineStories.Descriptions} />

### Children

#### Arbitrary content

Arbitrary `children` can be provided (eg. `<Badge />` component to display a state of something).

<Canvas of={ContextLineStories.Children} />

#### Badge

While Badges can be rendered using the `children` property, it's recommended to use the `badge` property instead.

<Canvas of={ContextLineStories.Badges} />

> If both `children` and `badge` are provided, the children will be rendered first.

### Icon

An optional icon can be provided through the `icon` property to be displayed at the left most side of the component.

> The icon will be rendered as a small StaticIcon.

<Canvas of={ContextLineStories.Icons} />
