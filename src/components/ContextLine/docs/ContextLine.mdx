## Usage

### Example

When using multiple `ContextLine` components, their padding and box-shadow will adapt according to their position in the parent component (first-child, default and last-child).
You can inspect it to see the effect.

<Canvas of={ContextLineStories.Example} />

### Label

A label can be provided in the `label` property. It's a required property.
<Canvas of={ContextLineStories.Label} />

### Children

You must add a **children** component (eg. `<Badge />` component to display a state of something).
