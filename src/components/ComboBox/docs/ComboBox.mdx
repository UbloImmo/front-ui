## Usage

`ComboBox` is a component that enables the user to select from a predefined set of options.
Uses [ComboBoxButtons](?path=/docs/components-forms-comboboxbutton-usage--docs) under the hood.

### Selecting only one or many options

By default, `ComboBox` allows the user to select a single option. To enable multiple selections, set the multi property to true.

<Canvas of={ComboBoxStories.MultiSelect} />

### Displaying the options

The options in `Combobox` are displayed in a row by default, wrapping into multiple rows on smaller devices. If you prefer a vertical display, you can set the direction property to `column`.

<Canvas of={ComboBoxStories.Direction} />

When needed, options can be displayed in a grid. To do so, provide the `columns` property.

> Options will be displayed in a grid with the specified number of columns.

<Canvas of={ComboBoxStories.Columns} />

### Disabling the options

To prevent the user from selecting an option, you can set the optional `disabled` property to `true` for the desired option.

<Canvas of={ComboBoxStories.DisabledOptions} />

### Showing the check icon

By default, the check icon displaying the selected option is visible. However, if you need a more compact layout of the `ComboBox` options, you can hide this icon by setting the `showIcon` property to false.

<Canvas of={ComboBoxStories.ShowIcon} />

### Creating more options

To enable the creation of new options, set the `creatable` property to `true`. An [ActionIcon](?path=/docs/components-actions-actionicon-usage--docs) will be displayed after the comboboxes.

You can also provide the desired callback for the creation of new options in the `onCreate` property.

<Canvas of={ComboBoxStories.Creatable} />

### Editing and deleting options

You can also enable the editing and deleting of options, by setting the `editable` and `deletable` properties to `true` for the desired options.

A [ContextMenu](?path=/docs/components-actions-contextmenu-usage--docs) will be displayed at the right of each option.

To customize the labels of the context menu items, you can provide the `optionEditLabel` and `optionDeleteLabel` properties.

As for the callbacks, use the `onOptionEdit` and `onOptionDelete` properties.

<Canvas of={ComboBoxStories.EditingAndDeletingOptions} />
