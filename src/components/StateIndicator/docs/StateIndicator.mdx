## Usage

**StateIndicator** is useful when displaying an entity's state or any other relevant page-wide information.

<Canvas of={StateIndicatorStories.Default} />

## Customization

### Labels

StateIndicator's `label` is a simple string. It is **required**.

> Please try to avoir excessively long label names as labels are set not to wrap onto new lines.
>
> <br />
> *These **will be clipped** if they overflow*.

<Canvas of={StateIndicatorStories.Labels} />

### Colors

There are a couple available colors, choosing one is as easy as setting the optional `color` property.

<br />
Defaults to `primary`.

<Canvas of={StateIndicatorStories.Colors} />

### Icons

StateIndicators internally use an [Icon](?path=/docs/components-icon-usage--docs) to render their provided `icon`.
As such, they can take any valid `IconName`.

> Please refer to the [Icon Repository](?path=/docs/components-icon-repository--docs) for a complete list.
