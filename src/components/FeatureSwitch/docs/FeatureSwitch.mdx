## Usage

`FeatureSwitch` is a component that allows you to toggle a feature on or off, or to select an option from a list corresponding to this feature.

## Variants

`FeatureSwitch` has 3 variants for the underlying input and the selection: `checkbox`, `switch`, `select`.

`FeatureSwitch`'s `variant` property is set to `switch` by default.

Each variant uses a different component as a selector/activator, along with their properties under the hood:

- `checkbox` uses [Checkbox](?path=/docs/components-forms-checkbox-usage--docs)
- `switch` uses [Switch](?path=/docs/components-forms-switch-usage--docs)
- `select` uses [SelectInput](?path=/docs/components-forms-input-selectinput-usage--docs)

## Disabled

`FeatureSwitch`'s `disabled` property is set to `false` by default.

## Additionnal properties, layout and content

### Compact layout

You can change the display of `FeatureSwitch` to a compact layout by setting the optional `compact` property.

<Canvas of={FeatureSwitchStories.Compact} />

### Description

To add explanatory content below the label of your `FeatureSwitch`, you can use the optional `description` property.

> The `description` property accepts both simple text strings and other content, including components from this library, HTML elements, or custom components.

This allows you to create entirely custom description within `FeatureSwitch`.

<Canvas of={FeatureSwitchStories.Description} />

### Tooltip

To provide help and tips about the feature to the user, you can use the optional `tooltip` property.

> uses [Tooltip](?path=/docs/components-feedbacks-tooltip-usage--docs) and its properties

<Canvas of={FeatureSwitchStories.Tooltip} />
