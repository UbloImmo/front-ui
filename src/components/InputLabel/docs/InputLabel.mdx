## Usage

**InputLabel** provides a label for the [Input component](?path=/docs/components-forms-input-usage--docs). It helps users understand the purpose of the input and makes a form more accessible by associating the label with the corresponding input field.

<Canvas of={InputLabelStories.Default} />

### Required

With the property `required` in **InputLabel**, you can visually indicate whether the input field is required. When the property `required` is true, an additional symbol (<Text color="warning-base">\*</Text>) is displayed after the label.

<Canvas of={InputLabelStories.Required} />

### Tooltips

Sometimes an input requires a detailed explanation. Provide an InputLabel with a `tooltip` property to display a tooltip.

> The `tooltip` property conforms to the [Tooltip component's properties](?path=/docs/components-forms-tooltip-properties--docs), minus its `children` property.
