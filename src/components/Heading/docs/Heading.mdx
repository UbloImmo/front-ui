## Usage

### Font size

You can use one of the 4 preset sizes according to your needs.

<br />
If no size is provided, defaults to `h1`.

<Canvas of={HeadingStories.Sizes} />

### Font weight

Control your heading's weight with the `weight` property. Defaults to `regular`.

<Canvas of={HeadingStories.Weights} />

### Font family

While the `font` property defaults to `sans` (Gilroy), you can also use `code` (DM Mono) if needed.

> Font weights and other properties are supported by all font families.

<Canvas of={HeadingStories.Fonts} />

### Italics

If you need to render italic headings (for some obscure reason), set this component's `italic` property to `true`.

<br />
It defaults to `false`, obviously.

<Canvas of={HeadingStories.Italic} />

### Decorations

Headings should be impactful.

<br />
If you feel your heading lacks in that regards, slap an&nbsp;<Text underline>
  underline
</Text>, <Text overline>overline</Text> or a <Text lineThrough>
  line through
</Text> it. Each decoration maps to its own property (`underline`, `overline`, `lineThrough`
respectively). Set some - or all - of them to `true` to see the result.
<br />
Defaults to `false`.

<Canvas of={HeadingStories.Decorations} />

### Colors

Headings inherit color support from our typography stack, though there should rarely be a need for a colored heading.
Still, below are some of the colors you could provide to a Heading.

<br />
