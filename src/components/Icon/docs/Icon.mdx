## Usage

### Sizing

Icons can take any CSS lenghts as their `size` property, as well as spacing labels (`s-4`, `s-5`, etc...).

> All sizes get converted to css `rem` values before being passed to the underlying `SVG` element.

## Available icons

All icons are generated to be able to keep up with new additions quickly.

Icons are selected using the `name` property. Each name corresponds to an unique Icon apart of :

- {objectKeys(BootstrapIcons).length} Bootstrap icons
- {objectKeys(CustomIcons).length} Custom icons
