## Usage

### Customizing colors

Icons can take any standard theme color as their `color` property. Here are a couple examples.

<Canvas of={IconStories.Colors} />

### Sizing

Icons can take any CSS lenghts as their `size` property, as well as spacing labels (`s-4`, `s-5`, etc...).

All sizes get converted to css `rem` values before being passed to the underlying `SVG` element.

<Canvas of={IconStories.Sizes}/>
