## Usage

`Hypertext` is used to render a clickable link that opens in a new tab, allowing users to navigate to different pages.

### Content

To label the redirection in the component, you can provide the desired text as the `children` property.

<Canvas of={HypertextStories.HypertextInText} />

### Colors

`Hypertext` default color is **primary**. To provide visual information on the nature of the link (error, warning, success), you can set the optional `color` property.

<Canvas of={HypertextStories.Colors} />

### Redirection

To provide the redirection link to `Hypertext`, use the `href` property.

> The link will open in a new tab

<Canvas of={HypertextStories.Redirection} />

### Accessibility

It is required to provide a title for `Hypertext`.
The provided text in the `title` property will be displayed by the browser when `Hypertext` is hovered, enhancing user experience and web accessibility.
