## Usage

`Hypertext` is used to render a clickable link that opens in a new tab, allowing users to navigate to different pages.

### children

To label the redirection in the component, you can provide the desired text as the `children` prop.

<Canvas of={HypertextStories.Default} />

### href

To provide the redirection link to `Hypertext`, use the `href` props.

> The link will open in a new tab
