## Usage

`EmailInput` component is an interactive single-line text-field, a specific type of [Input](?path=/docs/components-forms-input-usage--docs).
It is commonly used in form layouts to prompt users for their email address.

### Value

`EmailInput` is initialized without a value, enabling users to enter their email address.

When the users enter their email, `EmailInput`'s value is set to null until it matches the email adress pattern (eg: *mail@example.com*, *mail@example.adress.com*).
