## Usage

`TextInput` component is an interactive single-line text-field, a specific type of [Input](?path=/docs/components-forms-input-usage--docs), used to allow the user to enter text values and data.
It is commonly used in form patterns to prompt users for their name, address, and other details.

### Value

`TextInput` is initialized without a value, enabling users to enter their own data. It accepts all characters as text value.
