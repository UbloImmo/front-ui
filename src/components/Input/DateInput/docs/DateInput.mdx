## Usage

`DateInput` component is an interactive single-line text-field combined with a expandable Calendar, a specific type of [Input](?path=/docs/components-input-usage--docs).
It is commonly used in form layouts to prompt users for a date address.

### Value

`EmailInput` is initialized without a value, enabling users to enter their email address.

When the users enter their email, `EmailInput`'s value is set to null until it matches the email adress pattern (eg: *mail@example.com*, *mail@example.adress.com*).
