## Usage

`TypeSearchInput` component is an interactive single-line text-field, a specific type of [Input](?path=/docs/components-input-usage--docs).
It is mainly used to search and filter elements by typing a query in the input.

<Canvas of={TypeSearchInputStories.Query} />

### Value

`TypeSearchInput` is initialized without a value, enabling users to enter their query. It inherits from [TextInput](?path=/docs/components-input-textinput-usage--docs) and accepts all characters as text value.
