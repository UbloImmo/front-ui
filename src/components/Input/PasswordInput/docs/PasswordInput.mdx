## Usage

`PasswordInput` component is a specific type of the [Input Component](?path=/docs/components-input-props--docs), used to collect the user's password before logging in.

## Visibility

The default values of the `PasswordInput` component are hidden and encrypted for security. To reveal the input value, users can click on the icon located at the end of the input ( <Icon name="Eye" /> ), which set the `visible` property to true.
