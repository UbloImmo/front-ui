## Usage

`PasswordInput` component is a specific type of [Input](?path=/docs/components-input-usage--docs), used to collect the user's password before logging in.

## Visibility

The default values of the `PasswordInput` component are hidden and encrypted for security. To reveal the input value, users can click on the icon located at the end of the input ( <Icon name="Eye" /> ).

<Canvas of={PasswordInputStories.Visibility} />

The optional property `onVisibilityChange` allows you to provide a callback to be run when the password visibility changes.

If you want to disable password revelation, you can set the optional `allowCheckVisibility` property to false.
