## Usage

### Colors

StaticIcon takes **raw** colors _(e.g. `primary`)_ and **not** shaded colors _(e.g. `primary-light`)_ as its `color` property.

The resulting Icon's and background colors are computed based on this property.

<Canvas of={StaticIconStories.Colors} />

### Sizing

StaticIcons adapt their container's padding and border radius based on their `size` property.

Below are all the valid sizes for this component.

<Canvas of={StaticIconStories.Sizes} />

### Stroke

StaticIcons can display a thin stroke if provided with a `stroke` property.
Defaults to `false`

> Useful when you need just that bit more contrast.

<Canvas of={StaticIconStories.Stroke} />

### Icons

Through its `name` property, StaticIcon accepts any valid Icon name, and passes it directly to the underlying Icon component.

Please refer to the [Icon Repository](?path=/docs/components-icon-repository--docs) for a complete list.

<Canvas of={StaticIconStories.Icons} />

#### Indicators

StaticIcons can display an indicator, which is a small Icon positioned on the top right corner of the container.

<br />
The `indicator` property is an object that accepts an [icon name](?path=/docs/components-icon-repository--docs)
(`name`), a [palette color](?path=/docs/foundations-color-palette--docs) (`color`)
and an optional `tooltip` object that conforms to the [Tooltip](?path=/docs/components-tooltip-usage--docs)
component's props.
