## Usage

**InputAssistiveText** provides an optional descriptive text below the [Input component](?path=/docs/components-forms-input-usage--docs).

<Canvas of={InputAssistiveTextStories.Default} />

This text offers to users additional information about the input field.

<Canvas of={InputAssistiveTextStories.TextLength} />

> It can also notify users of any errors that might occur during input entry.

### Handling errors

You can indicate an error by setting the `error` property to `true` and providing the corresponding error text in the `errorText` property.

<Canvas of={InputAssistiveTextStories.States} />

### Customization

An optional icon can be displayed before the assistive - or error - text by setting the `assistiveTextIcon` property.

> If set to `true`, the default `SquircleInfo` icon will be displayed. Its color depends on the value of the `error` property.
