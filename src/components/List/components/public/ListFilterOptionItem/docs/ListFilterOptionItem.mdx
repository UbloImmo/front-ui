## Usage

Displays a single option, often in a ListFilter, while reflecting its selection state and configuration. Allows selection and deselection.

### Selection

#### Multi / Single selection

Some filters support multiple selections, while others only allow a single selection.
This behavior gets reflected in `ListFilterOptionItem`'s `multi` prop.

<Canvas of={ListFilterOptionItemStories.Multi} />

#### Selection states

Whether `multi` or not, a `ListFilterOptionItem` displays its `option`'s `selected` property.

<Canvas of={ListFilterOptionItemStories.Selections} />

### Behavior

Options come with a boat-load of behavior properties.

> For more information and details on what each property does, check out the [Option() method's documentation](?path=/docs/components-list-configuration-methods-option--docs).

#### Fixed

A fixed `ListFilterOptionItem` cannot ever be unselected.

<Canvas of={ListFilterOptionItemStories.Fixed} />

#### Disabled

A disabled `ListFilterOptionItem` cannot have its selection toggled, but can still be cleared.

<Canvas of={ListFilterOptionItemStories.Disabled} />

#### Highlight

Options may be highlighted by an external state, e.g. keyboard selection.
This is reflected in `ListFilterOptionItem`'s `highlighted` prop.

<Canvas of={ListFilterOptionItemStories.Highlighted} />

### Appearance

#### Label

`ListFilterOptionItem` displays its `option`'s `label` property.

<Canvas of={ListFilterOptionItemStories.Labels} />

> Option labels get truncated according to `ListFilterOptionItem`'s parent's CSS width.

#### Icon

`ListFilterOptionItem` displays its `option`'s `icon` property.

By default, it displays a `Circle` icon when the option is not selected, and a `CircleFill` icon when it is.

<Canvas of={ListFilterOptionItemStories.Icons} />

> Any valid icon names are supported. A complete list can be found [here](?path=/docs/components-iconography-icon-repository--docs).

#### Color

`ListFilterOptionItem` displays its `option`'s `color` property.

<Canvas of={ListFilterOptionItemStories.Colors} />

> Both `ColorKey` and `PaletteColor` types are supported when declaring an option's `color`. Each gets converted to the other and used as needed.

#### Hidden

When provided with a `hidden` property, the `ListFilterOptionItem` component will not render
