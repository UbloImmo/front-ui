## Usage

### Selecting an icon

`IconPicker` functions similarly to a single-select [ComboBox](?path=/docs/components-combobox-usage--docs), but is specifically designed for **icon selection**, making it useful for assigning a visual reference within a form context.

<Canvas of={IconPickerStories.Default} />

#### Choosing icons

By providing a list of `IconName`s to the `icons` property, one can customize which icons are avaiable to users.

<Canvas of={IconPickerStories.Icons} />

#### Rendering multiple rows.

The `icons` property also allows for a 2D array (`IconName[][]`) to be provided.

> Each inner list will be rendered as a row in the `IconPicker`.

<Canvas of={IconPickerStories.Rows} />

### Required

By default, users can clear their selection. To prevent this and make a selection mandatory, set the `required` property to `true`.

> Give it a try below, the selection remains even after clicking on the selected icon.

<Canvas of={IconPickerStories.Required} />

### Disabled

To disable user interaction with `IconPicker`, set the optional `disabled` property to `true`. This will prevent user from selecting an option or modifying an existing selection.

<Canvas of={IconPickerStories.Disabled} />
