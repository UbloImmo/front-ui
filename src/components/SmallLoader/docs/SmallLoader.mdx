## Usage

`SmallLoader` is an infinite looping animation loader as a progress bar. It gives users a visual indication while they wait for information or for the page they are redirected to.

`SmallLoader` utilizes the `Loading` component with the `ProgressBar` animation property, dynamically adjusting its width to match the wrapping element.
