## Structure
The library generates and exports its colors directly inside the global `theme` object.

- Each **color** is represented as an `object` that contains several **shades**
- Each **shade** contains both `rgba` and `hex` properties and an `opacity()` method to allow for further customization

```tsx
const color = {
  "[color]": {
    "[shade]": {
      "hex": "#fff",
      "rgba": "rgba(255, 255, 255, 1)",
      "opacity": (alpha: number) => `rgba(255, 255, 255, ${alpha / 100})` 
    }
  }
}
```

## Usage

Colors and shades can be used through various ways.

### Inside a React component

```tsx
// import the `useTheme` hook as usual
import { useTheme } from "styled-components";

const MyComponent = () => {
  // get theme from provider
  // access `primary` and any other color directly from `theme`
  const { primary } = useTheme();
  // access base color and `opacity()` method from `primary.base`
  const { rgba, opacity } = primary.base;
  // use the color
  const style = {
    // use primary.base at 50% opacity for the background...
    background: opacity(0.5),
    // ...and at 100% opacity for the text
    color: rgba,
  };

  return <div style={style}>Test</div>;
};
```

### Inside a regular CSS declaration

Colors are made available as regular CSS variables globally, under the name `[color]-[shade]`

This is also valid inside `styled-components` rulesets.

```css
.badge.success {
  background: var(--success-light);
  color: var(--success-dark);
}

.badge.primary {
  background: var(--primary-light);
  color: var(--primary-dark);
}

.button.disabled {
  background: var(--gray-400);
  color: var(--gray-600);
}
```
