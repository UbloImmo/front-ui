## Fonts

### Gilroy

Ublo's interfaces use `Gilroy` as the primary font, a modern sans-serif that's accessible and functional.
Various weights, sizes and letter spacings are combined in order to make it work in any setting.

<FontPreview font="sans" fontName="Gilroy" />

### DM Mono

In the rare cases where we need a monospace font, we use `DM Mono`.

> DM Mono's weight names have been shifted up to maintain consistency with the Gilroy weights.
> `light` is now `regular`, `regular` is now `medium` and `medium` is now `bold`.

<FontPreview font="code" fontName="DM Mono" />
